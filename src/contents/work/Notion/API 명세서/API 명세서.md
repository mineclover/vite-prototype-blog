## 속성

name(title) : 제목
Method : GET , POST , DELETE , PUT
URL : baseUrl 제외
Request : 요청 값
Response : 응답 값
검수인 : 확인 할 사람
담당자 : 작업자
Request Parameters : 응답 형식 , status + body , 쿠키 등
error ( Request Parameters ) : 응답 형식 , status + body , 쿠키 등
