## 소개

URL 에서 사용할 수 없는 예약어
`! * ' ( ) ; : @ & = + $ . , / ? # [ ]`
음 `-_%` 는 쓸 수 있는데 `%` 는 인코딩 시퀀스로 사용됨으로 실질적으로 , `-` 와 `&` , `~` ,`_` 를 사용할 수 있다는 것을 확인했다

`&` 는 중간에 쓸 수 있었지만, 이것도 서브 예약어에 포함되는 것 같다
